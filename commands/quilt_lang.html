class quilt_lang.DictObject(_dict)¶

Bases: object

Fancier way of converting nested dictionary to an object!

Parameters:_dict (dictionary) – Already defined dictionary 
>>> d = {'a': 1, 'b': {'c': 2}, 'd': ["hi", {'foo': "bar"}]}
>>> x = DictObject(d)
>>> x.b.c
2
>>> x.d[1].foo
'bar'
